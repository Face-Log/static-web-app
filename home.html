About Project

This is a software based project that deals with face recognition of a customer and displays relevant details such as count of new customers & repeated customers. Along with this it also displays the age, gender and emotions of the customer.

This product is suitable for any retail shop that regularly deals with customers. By using OpenCV, python and face recognition modules this software aims at providing reliable customer data. This product uses up to date technology to optimize its performance.

Face Detection

For Face detection we have used the concept of landmarks which will let you detect landmarks of detected face. Once the face will be detected it will be compared with the previous data. If the face is matched then it will be stored in 'known' folder and if not matched then it will be stored in the 'unknown' folder. By comparison it will thus inform whether the customer is repeated or not.

Data Analyses

We have used the pre-trained models that helps in the analysis of gender, age and emotions of the detected faces. We have written code that would predict the Age (Different age groups 11-15, 16-20,21-25 etc.), Sentiments (happy, sad, angry etc.), Gender and landmarks.

Database

We have used SQLite for database management and for visualising the data we have used DB Browser. As soon as the face will be detected, it will be stored in the database along with its details like unique ID, age, emotions and gender.We have created 3 tables - customer_master, location_master and visit_register. We have designed the schema in such a way that user can access the data easily.

Frontend Work

Our Frontend work includes the designing of website which can be used by the admins to see the statistical data of their customers. Along with the details of customer, the website includes the details about entire project and its developers.
